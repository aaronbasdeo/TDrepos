Understand custom fields for users

Navigation path

Manage > Channel > Settings > Custom UI | Custom Form Fields > (User Custom Attributes)

Your marketplace collects and stores certain details whenever a new user account is registered. When the standard set of collected details is not sufficient for your purposes, you can add UI fields to the user account registration workflow.

Later, to access these additional details, you can submit calls to the API endpoint that returns details about one user account specifically.

 NOTE:  When you lack access to this feature because access to it is not turned On, contact your AppDirect account representative.
