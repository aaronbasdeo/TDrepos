Desktop browser guidance

The companies that develop and distribute desktop browsers release updated versions of them continually, just as we develop and distribute updated versions of the AppDirect platform continually. Beyond this, your computer might use any of many possible operating system (OS) releases. The climate in which these systems and tools interoperate is as dynamic as it could possibly be. The number of possible permutations climbs every week.

So, while we pay close attention to making AppDirect work well in popular browsers on recent OS versions, only you can determine what browser and operating system are best for you. Table 1-1 exposes our priorities for testing, establishing, and maintaining platform performance and compatibility in the most popular browsers for Windows and Mac OS X client systems.
