Understand Network Manager

Part of the flexibility of the AppDirect platform is that it allows for — but never requires — a defined hierarchy of parent (corporate) and child (division or affiliate) marketplaces. When multiple AppDirect instances are all subordinate to central oversight by a corporate parent, our UI portal called Network Manager is where this oversight occurs.

Network Manager is the environment where corporate executives can oversee, override, and overrule what their channel administrators do on any affiliated, child marketplace.

The only AppDirect instances that include Network Manager are ones in the Network plan, and access to the Network Manager portal is restricted to the small subset of registered users who are authorized to see and use it. The relevant user role is corporate administrator.
