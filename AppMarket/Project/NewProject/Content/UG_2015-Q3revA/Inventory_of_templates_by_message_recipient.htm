Inventory of templates by message recipient

Tables in this section expose which templates are associated with each message recipient enumerator type.

channel admin

Messages target someone with the channel administrator user role. See Templates for the ‘channel admin’ enumerator.

company admin

Messages target Someone with the company administrator user role, whose company — or one of its associated users, ordered products, or payment methods — is the message topic. See Templates for the ‘company admin’ enumerator.

customer support

Messages target someone with the channel customer support user role. See Templates for the ‘customer support’ enumerator.

existing user

Messages target someone with any or no assigned user role, whose account on your marketplace has been activated. See Templates for the ‘existing user’ enumerator.

new user

Messages target someone with any or no assigned user role, for whom a user record exists, but whose account on your marketplace is not yet activated. See Templates for the ‘new user’ enumerator.

product developer

Messages target someone with the developer user role, whose product is the message topic. See Templates for the ‘product developer’ enumerator.

product owner

Messages target someone with any assigned user role, who is held accountable for complying with a contract’s end-user terms and conditions. Typically, this is either the user who created the relevant order or someone with the billing administrator user role, who has taken over responsibility for billing and contract compliance. See Templates for the ‘product owner’ enumerator.

product user

Messages target someone with any assigned user role, who has also been assigned to consume one of the described product’s allocated seats. See Templates for the ‘product user’ enumerator.

reseller

Messages target someone with the reseller user role. See Templates for the ‘reseller’ enumerator.

reseller manager

Messages target someone with the reseller manager user role. See Templates for the ‘reseller manager’ enumerator.
