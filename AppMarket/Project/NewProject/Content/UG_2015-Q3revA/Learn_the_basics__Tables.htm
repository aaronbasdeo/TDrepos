Learn the basics: Tables

UI tables gather and organize records about a given population, inventory, accumulation, collection, or other set of peers — such as users, companies, products, or orders. Any kind of broadly similar records from your marketplace might be pooled together into a UI table.

Each row in a table presents attributes about exactly one entity. Its attributes, when taken as a whole, can be broadly informative.

Each cell in a given row cites the value of one attribute. Cell values are sometimes linked to related information. In such a case, click the link in a table cell to load its related material in your browser.

Much of the UI on an AppDirect-powered marketplace is organized around and within tables. Topics in this section explain table UI patterns.
