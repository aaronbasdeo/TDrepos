Navigation path

UI fields and elements on the Reports page

Element

Description

Show Filters (button)

Exposes a banner above the table, where UI elements can help you to shape which records the table should represent — both in its visible rows and, optionally, in any CSV file that you generate and download.

Report Type (list)

Start date between (date picker)

End date between (date picker)

Created On (column)

The calendar date (relative to the physical location of the datacenter where your marketplace is hosted) when the corresponding report ran.

Report Type (column)

The high-level purpose and category of the corresponding report.

File Name (column)

The literal filename of the CSV file that is available to download.

Start Date (column)

A calendar date to limit the oldest data that the corresponding report cites.

End Date (column)

A calendar date to limit the newest data that the corresponding report cites.

(pagination controls)

Click these UI elements under the table to jump between respectively distant ranges of records in it, when it contains more records than it is configured to render at one time as visible rows.
