Work with your marketplace dashboard

Navigation path

Manage > Channel > Marketplace > Home | Dashboard

Procedure

Go to Manage > Channel > Marketplace.

Click through Home | Dashboard.

Do any of the following.

Would you like to change the reporting period that dashboard gauges represent?

Set the interval that all dashboard gauges describe

Click a named button on the time frame button bar to switch all of your dashboard gauges to match the corresponding duration.

Would you like to approve or deny vendor requests to add their network products to your catalog?

Consider requests to list network products.

Click Review Requests.

This button is visible only when at least one vendor request is pending. See Approve or deny product publication approval requests.

Would you like to check the worthiness of fully integrated products that are ready to publish?

Consider requests to publish native products

Click a product name in the The Following Apps are Waiting... area.

This area is visible only when at least one vendor request is pending. See Approve or deny product publication approval requests.

Would you like to resolve customer API integration events for your vendors?

Resolve any pending customer events for vendors

Click Review N pending events, where N is a count of pending events that await manual resolution.

This link is visible only when at least one customer event is pending. See Run and verify API integration tests.

Would you like to vet  the product questions, ratings, and reviews that users submit?

Approve and post user-generated content or decline to post it

Click Review N pending posts, where N is a count of user-generated posts that await vetting.

This link is visible only when at least one user post is pending. See Approve or deny customer-generated content.

Stop. You have completed this procedure.
