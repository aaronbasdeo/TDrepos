User roles can differ

Certain user roles are only ever enabled upon request. Certain other user roles are available for assignment only under narrow conditions. Still more user roles, which are enabled by default, can nonetheless be disabled. And some marketplaces implement additional, nonstandard user roles. Also:

The UI labels that specify user role names can be changed per marketplace instance, upon request.

The user roles that are assigned to you constrain which other user roles you can see and manage, if any.
