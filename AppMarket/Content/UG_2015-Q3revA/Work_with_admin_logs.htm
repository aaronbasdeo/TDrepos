Work with admin logs

Navigation path

Manage > Channel > Marketplace > Events | Admin Logs

Before you begin

Log in to your marketplace as its administrator.

Procedure

Go to Manage > Channel > Marketplace.

Click through Events | Admin Logs.

(Optional) Click Show Filters to expand a toolbar whose controls help you to filter which event records the table should show. Then, do any or all of the following.

 NOTE:  Filter selections take effect immediately, and each effect is additive.

Do you want to see log entries for just one administrative action type?

(Optional) Filter events by administrative action

Choose an option from the Show list.

Do you want to see log entries from a specific time frame?

(Optional) Filter events by when they occurred

Define the start and end of an order placement date range by entering dates manually in the paired date-entry fields.

OR

Click a field’s corresponding calendar icon () to expose the date-picker widget for populating that field.

 NOTE:  When you enter dates manually, use the calendar date notation syntax that is standard in your locale and which your marketplace is configured to use — either M/D/YY) or D/M/YY.

(Optional) Do any of the following.

Click a clickable product name to view details about the corresponding product.

Click a clickable company name to view details about the corresponding organization.

Click a clickable username to view details about the corresponding user account.

Stop. You have completed this procedure.
