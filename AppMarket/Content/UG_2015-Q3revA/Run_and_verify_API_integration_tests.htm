Run and verify API integration tests

Navigation path

Manage > Channel > Products > Catalog | API Tests

Procedure

Go to Manage > Channel > Products.

Click through Catalog | API Tests.

Browse, search, or filter the list of all integrated products until you surface the product whose API integration endpoints should be tested.

Click Test.

An animated progress indicator () replaces the Test button () temporarily, while tests run against the product’s API integration endpoints.

Color-coded badges — green, yellow, or red — indicate the health of each API endpoint.

Click a product name (in the App column) to load the page where you can check any of its integration settings in depth and correct whatever is demonstrably wrong.

Click Save.

Stop. You have completed this procedure.
