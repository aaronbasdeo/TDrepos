Table filtering patterns

Filtering is a technique for attributing high value to portions of a set that match for a certain trait, and low value to all other portions. It sifts undesirable portions from a set, exposing only what matters most to whomever defined the filter.

An AppDirect-powered marketplace is liable to provide more than one approach to filtering. In some cases, you might use check boxes to whitelist information subtypes that matter to you and, in other cases, you might choose options from one or more lists.
