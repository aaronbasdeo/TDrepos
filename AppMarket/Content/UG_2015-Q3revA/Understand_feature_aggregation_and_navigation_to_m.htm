Understand feature aggregation and navigation to manage products

Table 3-1 is a quick reference to the product management features of Marketplace Manager, exposing where we cluster such functions together and citing exactly where in the UI you can find each function.

Feature navigation to manage products


Manage > Channel > Marketplace >

Home |

Dashboard

Follow shortcuts to other pages, where you can do any of the following.

Judge the merit of customer-generated content before publishing or rejecting it.

Consider vendor requests to add network products to your marketplace.

Consider vendor requests to integrate products directly with your marketplace.

Resolve pending API integration events.

Reviews & Questions

Here, you can optionally judge the merit of customer-generated content before publishing or rejecting it.


Manage > Channel > Products >

Catalog |

Product Catalog

Use UI elements and features here, optionally, to do any or all of the following.

Add or remove products in your native product catalog (if you have one) or in the network catalog.

Approve or deny vendor-requested publication of new and updated products.

Filter, search, or browse the published products on your marketplace.

Modify marketplace placement and availability settings for any published product.

Manage use of downed API endpoints for any published product.

Adjust the sorting weight on listing pages for any published product.

{your brand} Products

Use UI elements and features here, optionally, to do any or all of the following.

Remove products from your native catalog, published or not.

Approve or deny vendor-requested publication of new and updated products in your native catalog.

Filter, search, or browse the products in your native catalog.

Open the product dashboard for any product in your native catalog, published or not.

API Tests

Use UI elements and features here, optionally, to check the operational status of every API endpoint for every product.

Promotions |

Bundles

Use UI elements and features here, optionally, to do any or all of the following.

Create a bundle.

Browse defined bundles.

View bundle details.

Publish a bundle.

Edit a bundle.

Discounts

Use UI elements and features here, optionally, to do any or all of the following.

Create a discount.

Browse defined discounts and, optionally, filter to include active or inactive discounts exclusively.

Search defined discounts.

Download a CSV file of the records that underlie your applied filter, if any.

View bundle details.

Edit a discount.

Delete a discount.

Credits

Create a credit.

Browse defined credits.

View credit details.

Edit a credit.

Delete a credit.

Product Groups

Use UI elements and features here, optionally, to do any or all of the following:

Browse defined categories, attributes, and customer groups.

Edit category settings. [Despite being simple, the UX here is not ideal.]

Create a category and configure its advanced settings, if any.

Edit a category, including any advanced settings.

Delete a category.

Create an attribute and configure its advanced settings, if any.

Edit an attribute, including any advanced settings.

Delete an attribute.

Create one customer group and edit it.

Add a media source.

Delete a media source.

Top Customers

Add a reference customer for vendors to cite.

Edit a reference customer.

Delete a reference customer.

Show or hide all product-specific questions.

Enforce manual approvals for all user-generated content before publication.
