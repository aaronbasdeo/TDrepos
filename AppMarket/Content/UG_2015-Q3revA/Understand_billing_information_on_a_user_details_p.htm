Understand billing information on a user details page

Navigation path:

Manage > Channel > Marketplace > Home | Customers > [Browse by Users] > {user}

When you click through a user record on the Users page, Marketplace Manager loads details about just that user. These details communicate:

Exactly how much money the user has spent thus far through your marketplace

Real-time counts of all the user’s:

free trials

expired free trials

purchased products

suspended products

unpaid invoices

In addition, a set of inline tables on each user details page helps you to search, filter, and track records of all orders, invoices, and payments that are associated with the described user.

Furthermore, a set of user management options help you to make purchases on behalf of the described user.
