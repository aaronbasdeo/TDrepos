Understand the sitewide default user role setting

Navigation path

Manage > Channel > Settings > Settings | Marketplace Functionality > Customer Roles | User’s Default Role

You can set which user role is preselected on the page where company administrators choose a user role to grant by default to their respective companies’ associated users.

Although you set which option is preselected for company administrators, each one of them is free to choose a different option than one that you set to be their default. Furthermore, they are free to assign different roles to their users than whichever role is their respective companies’ default.
