App Usage Logs page element reference

Navigation path

Manage > Channel > Marketplace > Events | App Usage Logs

UI fields and elements on the App Usage Logs page

Element

Description

Show Filters (button)

Exposes a banner above the table, where UI elements can help you to shape which records the table should represent — both in its visible rows and, optionally, in any CSV file that you generate and download.

Created between (date picker)

Download CSV (button)

Generates and downloads to your client system a CSV file whose records are the underlying basis for the App Usage Logs table as you filtered or otherwise shaped it.

Search (field)

Enter a text query string to run against all records in the table. Then, press Enter to submit the query.

Launched on (column)

The calendar date when the cited user opened the cited product.

Application (column)

The product that was opened.

Company (column)

The name of the organization whose associated user opened the product.

User (column)

The full name of the user who opened the product.

(pagination controls)

Click these UI elements under the table to jump between respectively distant ranges of records in it, when it contains more records than it is configured to render at one time as visible rows.
