When you develop editions of a product, you can create features that are common to all editions. These appear on the product profile page, where all editions are listed. In the same manner, you can create footnotes that apply to all editions.  

Following is an example of what your profile page might look like with shared features and footnotes.

Notes regarding the preceding image:

  • The features that apply to all editions are visible in the All Plans Include area in the top left corner of the page.
  • Any footnotes entered appear at the bottom of the page under Notes.

Use the following procedures to create shared features and footnotes.

Procedures

Step 1. Go to Developer > Products >  {productName} | Edit > Editions and Pricing | Shared Features & Footnotes. The following page appears.

Step 2. Under Shared Features, enter a shared feature in the empty Feature field. Click Add Footnote to add another feature. You may add as many features as required using the same process.

Step 3. Under Footnotes, enter a footnote to appear at the bottom of the product profile page. If needed, add additional footnotes.

Step 4. Click Save and Preview to see how the shared features and footnotes appear on the product profile page. Alternately, you may forego the preview step by immediately clicking Save after entering all features and footnotes.

Please note: When in preview mode, to view editions and shared features and footnotes, click the last tab on the right of the product profile page.
