After you create your product and have added features, you can manage those features. For example, you may want to add more features, or edit existing features.

Procedure

Step 1. Go to Manage > Billing and Distribution > Products > {Product name} | Edit > Profile & Branding | Profile | Features. This opens the list of existing features and also displays the Add New Feature button. 

 

Step 2. To edit an existing feature, click the line that displays that feature, and the Manage Feature page opens.

Note: If you want to add a new feature rather than manage an existing one, click Add New Feature, and complete the steps outlined in the Add features section.

Step 3. Edit the features as needed.

Step 4. After you complete your first feature, you have two choices:

a. Click Save.  This saves your changes and opens the Manage Feature page, from which you can further edit the feature or navigate to any page as required.

b. Click Save and Preview to open the Features tab of the product's Preview Profile page. 

All areas of the Features tab can be edited from this preview page. To make changes, click Edit Profile.

Using the mouse, go to the area you want to edit. Following are the the available feature management options from this page:

  • Click the pencil where the product name is listed—Opens the Manage Listing page.
  • Click the feature list on the left—A + symbol allows you to add another feature.
  • Click the pencil in the area containing the feature description—Opens the Manage Feature page.
  • Click the Previous and Next arrows in the area containing the feature description—Displays the previous or next feature, allowing you to manage those features from this area.
