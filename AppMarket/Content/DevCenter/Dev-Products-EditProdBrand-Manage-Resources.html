As a Developer, you can upload resource documents and videos to the product profile page. These can be useful in providing additional information about the product, such as how-to videos, white papers, FAQs, specification sheets, and other useful resources.

Resources can be uploaded as PDFs, Scribd documents, or as links to YouTube or Vimeo videos. You can select, for each resource, whether it is visible in the marketplace or only to the reseller.

Procedure

Step 1.  Go to Manage > Billing and Distribution > Products > {Product name} | Edit >, Profile & Branding | Resources. This opens the Manage Resources page.

Step 2. Under Documents, you have the option of adding PDF documents, Scribd documents, or both, as follows:

To upload a PDF

  • ​Enter the document name as you want it to appear on the Resource tab on the product profile page.
  • If appropriate, enter a document link. 
  • If the file is being uploaded from your computer, select Choose File, then locate and select the file on your computer.
  • If you want the file to be visible only to resellers, click Only visible to Resellers.
  • If you want to add another PDF resource, click Add PDF Document. 

  • Add as many PDF resources as required.

To upload a Scribd document

In the fields provided, enter the Scribd document's:

  • Document name
  • Document ID
  • Document key
  • If you want to add another Scribd document, click Add Scribd Document
  • Add as many Scribd documents as required.

Step 3. If you want to upload a video, complete the fields shown in the following image. 

  • Enter the name of the video as you want it to appear on the Resources tab of the product profile page.
  • Enter the YouTube or Vimeo link, as well as a description of the video (this is visible to users).
  • To add another video, click Add another video and follow the same procedure.
  • Add as many videos as required.
Note: For each resource type—pdf, Scribd document, and video— you can choose to make the file visible only to resellers by clicking Only visible to Resellers at the bottom of the section. 

Step 4.  After you have uploaded all required resources, take one of the following actions:

  • Click Save if you do not need to preview your changes. This saves your changes and returns you to the Manage Resources page.
  • Click Save and Preview to the Resources tab of the product profile page. This allows you to preview how your changes will appear to customers. Following is an example.

All areas of this page can be edited from here. To make changes, click Edit Profile. Using the mouse, go to the area you want to edit, and editing symbols become visible. 

  • Click the pencil in the product name area at the top to open the Manage Listing Info page.
  • Click the + symbol in the Resources list on the left side to add more resources.
  • Click the pencil in the main area displaying the resources you uploaded to open the Manage Resources page, where you can manage all resources.
