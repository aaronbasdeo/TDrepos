After you have created one or more add-ons, you can manage them by changing fields and options within the add-on product, or by cancelling the add-on. The procedure described on this page assumes you have created at least one add-on, and that you want to modify one or more elements that were previously-configured or cancel the add-on.

Change an add-on

Procedure

Step 1. Navigate to Developer > Dashboard | Products | {Product} > Edit. This opens the product configuration page. On the left side of the page, you can see the number of add-ons currently available for this product. There are currently two add-on offerings in this example.

Step 2. Click Add-on-Offerings to show the add-ons currently available. In the following example they are: Add user and Free domain name.

 

Step 3. Click the add-on offering for which you want to make changes. This opens the Manage Add-on page.

Step 4. Change options or fields as required to meet your new requirements.

Note: For a description of each of each field and option, see the Add add-on offerings section. 

Step 5. When you are finished making changes, you can choose one of two options:

  • Immediately click Save Add-on.
  • First preview the changes by clicking Save and Preview. When you are satisfied with your changes, click Save Add-on.

Clicking Save Add-on returns you to the Manage Add-on page. 

Cancel an add-on

To cancel an add-on, follow steps 1 to 3, and when you are on the Manage add-on page for the selected add-on, click Delete Add-on at the bottom of the page.

Note: When you click Delete Add-on, the add-on is removed immediately. There is no confirmation dialog box, so you should be sure you want to remove the add-on prior to clicking Delete Add-on.
